# Техническая документация проекта

## Общая информация

Проект представляет собой приложение для многокритериального анализа альтернатив (Multi-Criteria Decision Making - MCDM) с графическим интерфейсом на PyQt5. Приложение позволяет пользователям загружать данные из различных источников (базы данных SQLite, CSV/XLSX файлы), применять различные методы анализа альтернатив и получать ранжированные результаты.

## Архитектура приложения

### Графический интерфейс

Приложение состоит из следующих основных компонентов:

1. **App (QMainWindow)** - Главное окно приложения
2. **CalculateAlternativesWindow** - Окно для расчета альтернатив на основе данных из базы данных
3. **InputAlternativesWindow** - Окно для ручного ввода альтернатив или импорта из файлов
4. **AssessmentWindow** - Окно настройки параметров оценки и выбора методов
5. **ResultsWindow** - Окно отображения результатов анализа

### Поддерживаемые методы анализа

Приложение реализует следующие методы многокритериального анализа:

- **MINSUM** - Минимальная сумма
- **MINMAX** - Минимум-Максимум
- **MAXMIN** - Максимум-Минимум
- **DIP** - Расстояние до идеальной точки
- **TOPSIS** - Техника упорядочивания предпочтений по сходству с идеальным решением
- **WSR** - Weighted Sum Model
- **ELECTRE** - Метод многокритериальной оценки ELECTRE IV
- **VIKOR** - VlseKriterijumska Optimizacija I Kompromisno Resenje
- **ОРП** - Обобщенное решающее правило
- **AHP** - Метод анализа иерархий
- **CHP** - Метод анализа консенсуса

## Работа с данными

### Источники данных

Приложение поддерживает работу с двумя типами источников данных:

1. **Базы данных SQLite** - С возможностью визуализации связей между таблицами
2. **Файлы CSV/XLSX** - Для импорта табличных данных

### Визуализация результатов

Результаты анализа отображаются в виде:
- Таблиц с ранжированными альтернативами
- 3D-графиков для визуализации зависимостей между критериями
- Возможности экспорта результатов в Excel

## Структура кода

Файл `main.py` содержит все классы и функции приложения:
- Классы графического интерфейса
- Функции реализации методов анализа
- Вспомогательные функции для работы с данными
- Классы для визуализации графиков

## Технологии

- Python 3
- PyQt5 для графического интерфейса
- NumPy для численных вычислений
- Pandas для работы с данными
- Matplotlib для построения графиков
- SQLite для хранения данных
- OpenPyXL для работы с Excel файлами
